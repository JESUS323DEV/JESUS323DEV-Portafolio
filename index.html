{ AtenciaHomes }

Atencia Homes es una web inmobiliaria desarrollada con enfoque en diseño limpio y navegación intuitiva.Combina estructura con Bootstrap, estilos personalizados y un sistema visual adaptable para distintos dispositivos.Fue creada como proyecto real para un cliente, cuidando tanto la presentación como la funcionalidad en cada página.

( Game-lab )

Plataforma donde reúno varios minijuegos desarrollados con JavaScript puro. Cada juego tiene su propia lógica, interfaz y mecánicas únicas, creadas desde cero para practicar programación interactiva y diseño dinámico. Es un proyecto en constante evolución, donde sigo probando ideas, efectos y mejoras visuales.

[ FormLab ]

Espacio dedicado a la creación y prueba de formularios inteligentes con validaciones en tiempo real. Combina HTML, CSS y JavaScript para ofrecer una experiencia dinámica, accesible y visualmente cuidada. Cada formulario está pensado para practicar lógica, usabilidad y diseño adaptable.Con el objetivo de perfeccionar la experiencia del usuario y la precisión del código.
